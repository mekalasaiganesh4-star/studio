# **App Name**: WasteLess Feast

## Core Features:

- Food Listing Upload: Hosts (hotels, hostels, kitchens) upload food details, images, testing reports, expiry time, and availability status.  Admin approval required.
- Food Browsing & Filtering: Users browse available food by location, type, price, and expiry time. Real-time availability updates are shown.
- Order Placement: Users place orders for home delivery or self-pickup, selecting quantity and preferred time.
- Payment Processing: Enable secure payment processing via COD, UPI, debit card, and credit card. 'Pay at Hotel' for self-pickup is supported.
- Admin Dashboard: Admins approve/reject listings, verify reports, manage users/hosts, monitor expiry, and view analytics.
- Animal Feed Redirection: Expired or surplus food is automatically moved to an Animal Feed section, available for farmers/NGOs. Pricing can be very low or free.
- AI-powered Food Suggestion Tool: Based on user preferences and past orders, suggest available food items using AI.

## Style Guidelines:

- Primary color: HSL 120, 50%, 50% - Forest green (#40BF40) for eco-friendliness and freshness.
- Background color: HSL 120, 20%, 95% - Very light green (#F0FAF0) to provide a clean backdrop and evoke nature.
- Accent color: HSL 90, 60%, 40% - Lime green (#50A828) to highlight key interactive elements.
- Body and headline font: 'PT Sans', a humanist sans-serif for headlines and body text.
- Use food-related icons, emphasizing freshness and safety, in a consistent style throughout the app.
- Employ a clean, card-based layout with clear distinctions between food items, focusing on essential details like price, expiry time, and source.
- Subtle transitions and animations to enhance user experience, such as smooth loading indicators and item reveal effects.
